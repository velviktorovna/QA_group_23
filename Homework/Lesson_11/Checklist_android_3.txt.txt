1. Установка

1. Установка из дистрибутива происходит без уведомлений об ошибках.
2. Успешно выполняется установка из магазина приложений.
3. Установка обновлений не вызывает ошибок.
4. Отмена установки происходит корректно, с удалением всех следов приложения.
5. Повторная установка после отмены возможна и выполняется успешно.
6. При попытке установки на неподдерживаемое устройство/версию ОС появляется предупреждение о несовместимости, и установка корректно завершается.


2. Запуск и выход из приложения

1. Приложение запускается при клике по его иконке.
2. Приложение запускается в режиме разделенного экрана
3. Приложение запускается при клике по уведомлению от него.
4. Приложение запускается по ссылкам из других приложений.
5. Приложение запускается по ссылкам из других приложений.
6. Приложение запускается по голосовой команде.
7. Приложение запускается при автозагрузке.
8. Приложение запускается при управлении жестами.
9. Приложение запускается при открытии ассоциированных файлов.
10. Приложение возобновляет работу после перевода в фоновый режим.
11. Время запуска приложения не слишком велико.
12. Возможен выход из приложения штатным способом.
13. Возможен выход из приложения при нажатии на кнопку «домой» (приложение переходит в фоновый режим).


3. Удобство использования

1. Управление приложением интуитивно понятно.
2. Навигация в приложении соответствует требованиям, предъявленным заказчиком
3. Приложение корректно обрабатывает смену ориентации экрана.
4. Приложение корректно обрабатывает изменение масштаба отображения.
5. Приложение корректно обрабатывает жесты multitouch.
6. Вызов клавиатуры происходит корректно, клавиатура не скрывает элемент страницы, в который вводится текст.
7. Элементы управления соответствуют выполняемым функции.
8. Расход заряда батареи соответствует нагрузке, создаваемой приложением.
9. Задержки переходов и/или открытия не критичны.
10. Приложение соответствует требованиям удобства использования для платформы
11. Нет «вылетов» приложения и/или неожиданно возникающих всплывающих окон.
12. (если заявлено в ТЗ) В приложении присутствует помощь для новых пользователей


4. Вёрстка (на всех заявленных в ограничениях устройствах и разрешениях)

1. Отсутствуют сдвиги элементов страницы (расположение элементов соответствует заявленному в макете).
2. На страницах нет обрезки текста.
3. На страницах присутствуют все заявленные элементы.
4. Элементы не пересекаются и не «наслаиваются» друг на друга.
5. Ни один элемент не выходит за границы экрана.
6. Отсутствуют артефакты адаптивного дизайна.


5. Обновления


1. Обновления устанавливаются корректно.
2. При запуске происходит проверка обновлений, и пользователю выдается информация об ограничениях текущей версии.
3. Появляются уведомления при нескольких пропущенных обновлениях.
4. Появляются уведомления при пропущенных критических обновлениях, препятствующих эксплуатации приложения.
5. Приложение работает после обновления ОС.
6. Приложение работает после прерванной или отмененной установки обновления.


6. Прерывания

1. Прерывание работы приложения телефонным звонком обрабатывается корректно.
2. Прерывание работы приложения получением СМС обрабатывается корректно.
3. Прерывание работы приложения сменой ориентации экрана обрабатывается корректно
4. Прерывание работы приложения блокировкой/разблокировкой экрана обрабатывается корректно.
5. Прерывание работы приложения системными уведомлениями обрабатывается корректно.
6. Прерывание работы приложения разрывом соединения с интернетом обрабатывается корректно.
7. Возможна работа приложения при слабом/нестабильном соединении (в зависимости от требований).
8. Возможна работа приложения в режиме энергосбережения.
9. При смене сети работа приложения не прерывается.
10. Работа приложения не зависит от процессов передачи данных в фоновом режиме.


7. Безопасность

1. В магазине приложений нет алертов на безопасность приложения.
2. При запуске приложения на устройстве не возникает коллизий (при использовании наиболее популярных антивирусных систем).
3. При установке и использовании приложение запрашивает и может использовать нужные разрешения.
4. Приложение корректно работает с пользовательской сессией (согласно требованиям).
